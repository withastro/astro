* Org mode in Astro

This page is authored in =.org= and compiled by =@astrojs/org-mode=.

- Native Org headings and lists
- Source blocks and tables
- JSX exports and Astro component support
- Custom tag mapping demo: [[/custom-components][replace =h1= with =Title= component]]

#+begin_quote
Org syntax stays readable, but you can still drop down to JSX when needed.
#+end_quote
