To support `.vue` files you will need the [Vue Plugin](https://github.com/snowpackjs/snowpack/tree/main/plugins/plugin-vue), which uses Vue 3.

```js
// snowpack.config.json
"plugins": ["@snowpack/plugin-vue"]
```
