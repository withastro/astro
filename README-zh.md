## 安装

**推荐**通过运行以下命令来安装最新版本的 Astro：

```bash
npm create astro@latest
```

你也可以**手动**安装 Astro，运行以下命令：

```bash
npm install astro
```

需要帮助？请从我们的[入门指南](https://docs.astro.build/en/getting-started/)开始。

想要快速查看示例？直接在浏览器中[打开一个入门项目](https://astro.new/)。
