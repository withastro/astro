# Astro 中文文档

现代网站构建框架。

## ✨ 特性

- ⚡ 零 JS 默认
- 📦 集成生态
- 🔌 支持 React/Vue/Svelte/...
- 📱 静态生成 + SSR
- 🌍 国际化支持

## 🚀 快速开始

```bash
# 创建项目
npm create astro@latest

# 启动开发服务器
npm run dev
```

## 📖 示例

```astro
---
const title = "欢迎"
---
<html>
  <head><title>{title}</title></head>
  <body>
    <h1>{title}</h1>
  </body>
</astro>
```
