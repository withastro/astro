# Contributing

## Development workflow

In the monorepo first install and build Astro:

```shell
yarn install
yarn build
```

To start the development server run:

```shell
yarn dev:vscode
```

Then in the **Debug** panel select **Launch Extension** from the dropdown and click the run button.
